# Spring Boot with MySQL

## STEPS FOR THIS SPRING BOOT APP
- Config JDK 11.
- Create recursos_propios database
- Build the application.
- Up application.

## BUILD the application
mvn clean install

## Up application.
Run Main class

## Access to swagger console
http://localhost:8080/swagger-ui.html
